# base-react-front
Este é o front-end da aplicação base-nodejs.
O back-end para esse projeto se encontra [aqui](https://github.com/FernandoOLI/base-nodejs)

## Instalação
Para instalar o projeto execute:

```bash
npm i
```

## Executar

```bash
npm start
```

## Requisitos
Npm, NodeJs, Idle de sua preferência
